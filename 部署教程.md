# 🌊 宇辰的漂流瓶 — 新手零基础部署教程

> 整个过程大约需要 30-40 分钟，不需要任何编程基础，按步骤复制粘贴即可。

---

## 📋 你需要准备的东西

- 一台电脑（手机操作不方便）
- 一个邮箱（用来注册账号）
- 一个163邮箱（用来发验证码给粉丝，也可以用QQ邮箱）

---

## 第一步：注册 Laf 云开发平台

**Laf 是什么？** 一个国内可以访问的免费云开发平台，可以帮你运行后端代码和存储数据，完全不需要自己买服务器。

1. 打开浏览器，访问：**https://laf.run**

2. 点击右上角「注册」，用手机号注册账号

3. 注册完成后，登录进入控制台

4. 点击「新建应用」→ 输入应用名称（比如：yuchen-bottle）→ 选择「免费版」→ 点击「创建」

5. 等待大约 30 秒，你的应用就创建好了

6. 在应用列表里，**找到并记下你的 App ID**
   - 它是一串字母数字，比如：`abcdef123456`
   - 你的后端地址就是：`https://abcdef123456.laf.run`

---

## 第二步：配置163邮箱（用来发验证码）

> 如果你没有163邮箱，可以去 163.com 免费注册一个

1. 登录你的163邮箱（https://mail.163.com）

2. 点击右上角头像 → 「设置」→ 「POP3/SMTP/IMAP」

3. 找到「SMTP服务」，点击「开启」

4. 按提示验证手机号，然后系统会给你一个「**授权码**」（不是邮箱登录密码！）

5. **把授权码保存好**，待会要用

---

## 第三步：在 Laf 设置环境变量

> 环境变量就像是保险箱，把敏感信息（邮箱密码）安全存放，不写在代码里

1. 在 Laf 控制台，进入你的应用

2. 左边菜单点击「环境变量」或「设置」→「环境变量」

3. 依次添加以下三个变量（点「新增」，填写名称和值）：

   | 变量名称 | 变量值（填你自己的） |
   |---------|-------------------|
   | MAIL_USER | 你的163邮箱地址，如：abc@163.com |
   | MAIL_PASS | 刚才得到的163授权码 |
   | IDOL_SECRET | YuChen2024!（或者你自己设定的密码） |

4. 保存环境变量

---

## 第四步：创建数据库集合

> Laf 内置 MongoDB 数据库，你需要手动创建3个"集合"（相当于3张表）

1. 左边菜单点击「数据库」

2. 点击「新建集合」，依次创建以下3个集合（名称要一字不差）：
   - `users`（存用户信息）
   - `bottles`（存漂流瓶内容）
   - `verify_codes`（存验证码，会自动清理）

3. 三个集合都创建完，看到列表里有这三个就对了

---

## 第五步：安装 nodemailer 依赖包

> nodemailer 是用来发邮件的工具包，需要提前安装

1. 在 Laf 控制台左边菜单点击「依赖管理」或「NPM 包」

2. 搜索 `nodemailer`

3. 点击安装，等待安装完成（大约1-2分钟）

## 第七步：修改前端 HTML 文件

现在要把前端 HTML 里的配置改成你真实的信息

用任何文本编辑器打开 `index.html`（记事本也可以）

找到这一段（在文件底部 `<script>` 区域）：

```javascript
const CFG = {
  API:       'https://YOUR_APP_ID.laf.run',  // ← 改这里
  IDOL_USER: 'idol_qiuyuchen',
  IDOL_PASS: 'YuChen2024!',                  // ← 改成和环境变量 IDOL_SECRET 一样的值
  DEMO:      true,                           // ← 改成 false
}
```

**需要修改的地方：**

1. 把 `YOUR_APP_ID` 换成你在第一步记下来的 App ID
   - 例如改成：`'https://abcdef123456.laf.run'`

2. `IDOL_PASS` 改成你在第三步设置的 `IDOL_SECRET` 的值（保持一致）

3. `DEMO: true` 改成 `DEMO: false`

保存文件。

---

## 第八步：把 HTML 文件放到网上

有两个简单方案，选一个：

### 方案 A：用 Laf 的静态托管（推荐，最简单）

1. 在 Laf 控制台左边菜单找「静态托管」或「文件存储」

2. 上传你修改好的 `index.html` 文件

3. Laf 会给你一个访问链接，把这个链接分享给粉丝就行了

### 方案 B：用 Gitee Pages（免费）

1. 去 https://gitee.com 注册账号

2. 新建一个仓库（选"公开"）

3. 把 `index.html` 上传进去

4. 进入仓库设置 → 找到「Gitee Pages」→ 点击「启动」

5. 几分钟后，你会得到一个 `xxx.gitee.io/xxx` 的网址

## ❓ 常见问题

**Q：粉丝说收不到验证码？**
A：检查第二步的邮箱设置，确认 SMTP 已开启，授权码填写正确。也提醒粉丝检查垃圾邮件文件夹。

**Q：点击"发送验证码"没反应？**
A：打开浏览器控制台（F12），看有没有红色错误信息，检查 CFG.API 地址是否填写正确。

**Q：云函数报错"无法连接邮件服务器"？**
A：确认 MAIL_USER 和 MAIL_PASS 环境变量填写正确，163邮箱的 SMTP 服务已开启。

**Q：想换成QQ邮箱发验证码？**
A：把 `send-code.js` 里的 `host: 'smtp.163.com'` 改成 `host: 'smtp.qq.com'`，port 改成 `587`，secure 改成 `false`。QQ邮箱同样需要去设置里开启 SMTP 并获取授权码。

**Q：如何修改偶像的登录密码？**
A：在 Laf 环境变量里修改 `IDOL_SECRET` 的值，同时修改 `index.html` 里 `CFG.IDOL_PASS` 为相同的值。

**Q：免费版 Laf 有什么限制？**
A：免费版每月有一定的请求次数和存储限制，对于粉丝互动的小应用完全够用（每月约10万次请求）。如果粉丝特别多再考虑升级付费版。
